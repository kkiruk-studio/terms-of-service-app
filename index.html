Pinclip 데이터 처리

URL 공유 및 메타데이터 수집

사용자가 iOS 공유 시트를 통해 Pinclip으로 URL을 전달하면, 앱은 해당 URL의 오픈그래프(OGP)/oEmbed 메타데이터(제목, 썸네일 이미지 URL 등)를 가져와 사용자의 기기에 저장합니다. 해당 요청은 원본 웹사이트(YouTube, TikTok 등) 또는 각 플랫폼의 공개 API로 직접 전송되며, 당사의 서버를 경유하지 않습니다.

사용자는 저장된 URL, 제목, 썸네일, 메모, 태그, 폴더 정보를 언제든지 수정 또는 삭제할 수 있습니다. 사용자는 저장하는 콘텐츠에 대한 법적 책임을 지며, 불법 콘텐츠 또는 제3자의 저작권·초상권·프라이버시 등을 침해하는 콘텐츠의 저장·공유 용도로 앱을 사용해서는 안 됩니다.

iCloud 동기화

Pro 구독자의 저장 데이터(폴더, 클립, 썸네일, 메모, 태그)는 사용자의 Apple iCloud 계정 내 private CloudKit 데이터베이스에 저장 및 동기화됩니다. 해당 데이터는 Apple의 인프라에서 암호화되어 전송·저장되며, 당사는 이에 접근할 수 없고 당사 서버로 전송되지 않습니다.

무료 플랜에서는 기기 내 로컬 저장소에만 데이터가 저장되며, 기기 간 동기화가 이루어지지 않습니다. iCloud 로그인 상태, 저장 공간, 네트워크 환경에 따라 동기화가 지연되거나 실패할 수 있으며, 이에 대한 구체적인 정책은 Apple의 iCloud 약관을 따릅니다.

앱은 날씨 데이터를 제공하기 위해 기기의 위치 정보를 사용합니다. 위치 정보는 서버에 저장되지 않으며, 날씨 API 요청 시에만 일시적으로 사용됩니다. 위치 접근 권한은 iOS 설정에서 언제든지 변경할 수 있습니다.

YouTube API 이용 (Sidefeed)

Sidefeed는 YouTube Data API v3를 사용합니다. Google OAuth 로그인을 통해 사용자의 좋아요 영상 및 구독 채널 정보에 접근합니다. 이 데이터는 관심 카테고리 분석에만 사용되며, 당사 서버에 저장되지 않습니다.

Sidefeed의 YouTube API 사용은 YouTube 서비스 약관 및 Google 개인정보 보호방침의 적용을 받습니다.

YouTube API 일일 할당량 제한에 따라 새로고침 횟수가 하루 10회로 제한됩니다.

음성 녹음 및 인식 (Talk Memo)

Talk Memo는 Apple Watch 마이크를 통해 음성을 녹음하고, iPhone의 온디바이스 음성 인식(Apple Speech Recognition)을 사용하여 텍스트로 변환합니다. 녹음 및 변환된 텍스트는 기기 내에만 저장되며, 외부 서버로 전송되지 않습니다. 음성 인식의 정확성은 보장되지 않습니다.

사용자는 녹음 내용에 대한 법적 책임을 가지며, 제3자의 동의 없이 대화를 녹음하여 저장하는 등 관련 법령에 위반되는 용도로 사용해서는 안 됩니다.
